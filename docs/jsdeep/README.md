## 种一棵树最好的时间是十年前，其次就是现在。


1. “苦难既然把我推到了悬崖的边缘，那么就让我在这悬崖的边缘坐下来，顺便看看悬崖下的浏岚雾霭，唱支歌给你听。”

   ———史铁生

2. “这些年一直在提醒自己一件事，千万不要感动自己，大部分人看似努力，不过是愚蠢导致的。

   什么熬夜看书到天亮，连续几天只睡几小时，多久没放假了。

   如果这些东西也值得炫耀，那么富士康流水线上任何一个人都比你努力多了。

   人难免天生有自怜情绪，唯有时刻保持清醒，才看得清真正的价值在哪里。”

   ———于宙《我们这一代人的困惑》

3. “迷失的时候，选择更艰辛的那条路。”

   ———松浦弥太郎

4. “在这邪恶的世界上，没有什么能够永久，即使是我们的麻烦也不能。”

   ———查尔斯·卓别林

5. “现今还有谁会照顾谁一辈子，那是多沉重的一个包袱。所以非自立不可。”

   ———《不易居》

6. “纵有疾风来，人生不言弃。”

   ———瓦雷里《海滨墓园》

7. “人生到处知何似，应似飞鸿踏雪泥。”

   ———苏轼《和子由渑池怀旧》

8. “撑住，才有后来的一切。”

9. “愿你年少足风流，一万星辰掬在手，三千清诗唱不休。

   愿你霜尘梦不朽，也有白月牵衣袖，也有春秋抚眉头。”

10. “在失意时终会明白，有些热爱如若无法代替，那么有些坚强必将绝处逢生。”

11. “文艺青年不行，还是太软弱了，缺乏一种骨子里的韧劲。太颓丧，太羞怯，毫无野性和心机，也没有杀伐决断的勇气。这样的人，其实是很难获得世俗层面的圆满和成功的。要想过得好，还是要摒弃掉骨子里的软弱和丧。积极起来，俗一点，糙一点，反而好。”

12. “真好，还没把生活过成一滩死水，还是会因为小事儿特别高兴，还是会因为几句话就想哭想笑的。”

13. “你哭吧，你应该哭。把砸烂的门扔在暗角里，让撕碎的书泡在雨水里，将从前的爱恨抛出去扔给饥饿的人们，看他们争先恐后，你就收起哭声，离开人群里的每一个自己。”

14. “天黑了就是遇上挡不住的大难了，你就得认命。认命不是撂下，是咬着牙挺着，挺到天亮。天亮了就是给你希望了，你就赶紧起来去往前走，有多大的劲儿就往前走多远，老天会帮你。孩子，你记着，好事来了她预先还打个招呼，不好的事咣当一下就砸你头上了。从来不会提前通知你！能人越砸越结实，不能的人一下就被砸倒了。”

   ———倪萍《姥姥语录》

15. “天黑了，谁能拉着太阳不让它下山？你就得躺下，孩子，不怕，多黑的天到头了也得亮。”

   ———倪萍《姥姥语录》

16. “我念的经，只有四个字:“人生苦短”。因为这苦和短，我马不停蹄，一意孤行。”

   汪曾祺《人间草木》

17. “我的心就如同这张面孔/一半纯白/一半阴影/我可以选择让你看见/也可以坚持不让你看见/世界就像是个巨大的马戏团/它让你幸福/却让我惶恐/因为我知道散场后永远是/有限温存/无限心酸。”
